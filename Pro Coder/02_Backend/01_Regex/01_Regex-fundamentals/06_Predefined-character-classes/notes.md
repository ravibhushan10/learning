Predefined/Shorthand character classes

<!-- ******************** -->
\d -> its equivalent to [0-9] match any digit
\D -> its equivalent to [^0-9] match any non-digit
\w -> matches any word character (equivalent to [A-Za-z0-9_])
\W -> matches non-word character (equivalent to [^A-Za-z0-9_])
\s -> matches any whitespace character(tabs,space,linebreak)
\S -> matches any non-whitespace character(tabs,space,linebreak)
